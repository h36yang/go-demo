# go-demo
Practice materials for Go

### Language Characteristics
* Fast Compilation
* Fully Compiled
* Strongly Typed
* Concurrent by Default
* Garbage Collected
* Simplicity as a Core Value

### What Go is good at?
* **Web Services**
* **Web Applications**
* Task Automation
* GUI / Thick Client
* Machine Learning

### Sample Web Service Project Structure
```
module "demo/webservice"
    package "models"
        struct "User"
```
